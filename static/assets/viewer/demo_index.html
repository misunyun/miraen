기능 안내

이 사이트는 엠티처 전자저작물 콘텐츠를 살펴볼 수 있는 사이트입니다.


1. 그리기 기능 - 그리기, 텍스트 입력, 도형 삽입, 지우개, 삭제, 가리기 기능을 사용할 수 있습니다.

2. 수업 도구 기능 - 판서, 타미어, 깜깜이, 주의집중, 발표자선정, 모둠점수판 기능을 사용할 수 있습니다.

3. 페이지 이동 - 좌/우 버튼으로 페이지를 이동하거나 목차의 링크를 이용하여 페이지를 이동할 수 있습니다.

4. 확대/축소 - 하단의 버튼을 통해 콘텐츠 확대/축소를 할 수 있습니다.

5. 설정 - 스킨 컬러 변경, 콘텐츠 화면 비율 변경을 할 수 있습니다.

6. 접기/펴기 - 하단의 버튼을 통해 목차를 펴고 접을 수 있습니다.
